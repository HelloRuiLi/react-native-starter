# react-native-starter

## 创建 react native app

1. npm install -g react-native-cli
 
2. react-native init ReactNativeStarter
    
    cd ReactNativeStarter
    
> xcode 必须大于等于 7.0 版本
     
## IOS 启动

npm install 

npm run ios
   
npm start
 
## Android 启动

安装Android Studio 以及对应的 Android SDK 和 Android 模拟器镜像文件 

npm run android
